Project Overview
This TypeScript app allows a restaurant to manage its menu efficiently.
It includes separate screens for chefs and guests, displays menu information, and uses loops, functions, and arrays to organize and manage data.
Learning Outcomes
At the end of this project, I am able to:
Use a for loop, while loop, and for-in loop in TypeScript to solve programming problems.
Define and use functions to organize my code.
Work with global variables to store and access data.
Use arrays to manage and display menu items.
Refactor code into multiple files for better structure and readability.
